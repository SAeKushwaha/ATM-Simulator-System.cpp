# ATM Simulator System

A console-based ATM simulation project built using C++ with dynamic memory allocation and pointer arithmetic.

## Features

- Deposit money with a one-time deposit limit.
- Withdraw money.
- PIN verification system.
- View transaction history.
- Check account balance.

## Concepts Used

- Dynamic Memory Allocation
- Pointer Arithmetic
- Arrays for storing transaction history
- Menu-driven programming
- Input validation to avoid unnecessary user prompts after an incorrect PIN

## Purpose

This project was built while learning C++ pointers and dynamic memory management without using array indexing or STL containers such as `std::vector`.
